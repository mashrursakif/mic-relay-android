# Mic Relay - Android Client

Android app for capturing device microphone input and streaming to PC, built with Android Studio

## Installation
Download latest version from [Releases](https://github.com/mashrursakif/mic-relay-android/releases/download/v1.0.0/app-debug.apk)

## Usage
- Connect android device on same WiFi network as PC
- Enter PC IP address and select port
- Click Start to start streaming microphone input to PC

This app requires microphone permission.
